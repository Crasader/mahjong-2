# OEasy

这个项目，有幸得到你的浏览。

这是一个基于 Qt 5 开发的项目，Qt Creator 4 作为主要的开发 IDE

使用前请在pro文件中添加 C++ 11 属性

主要目的是完成一款接口极简的工具，即插即用。

在极简的基础上不断细化，添加一些细致的功能，为优质的软件添加一份助力。

一个优秀的项目，少不了各位的支持与参与，欢迎你加入我们。

`注：如果我们的项目有幸得到您的使用，请告诉我们，我们会把贵公司的软件公布到 Wiki 中。`
